## Deployment

This site is automatically served via GitHub Pages from the main/master branch.

To enable GitHub Pages:
1. Go to your repository Settings
2. Navigate to "Pages" section
3. Under "Source", select the branch (e.g., `main` or `copilot/create-static-valentines-website`)
4. Select `/ (root)` as the folder
5. Click Save

Your site will be live in a few minutes!
